# Sleep Art Gallery

睡眠時間をジェネラティブアートに変換するWebアプリケーションです。
毎日の睡眠を記録すると、睡眠データに基づいたユニークな抽象アート作品が自動生成されます。

## デモ

`sleep-art-gallery/index.html` をブラウザで開くだけで動作します（サーバー不要）。

## 機能一覧

### 睡眠記録
- 「おやすみ」ボタンで就寝時刻を記録
- 「おはよう」ボタンで起床時刻を記録
- 睡眠時間を自動計算・表示

### 目覚まし機能
- 就寝時にアラーム時刻を設定可能
- Web Audio API による穏やかなメロディで起床
- ブラウザ通知にも対応

### ジェネラティブアート生成
睡眠データに基づいて、毎日ユニークなアート作品を Canvas API で生成します。

| 睡眠データ | アートへの影響 |
|-----------|--------------|
| 就寝時刻 | カラーパレット（早寝=青紫系、遅寝=暖色系） |
| 睡眠時間 | 要素の数・複雑さ |
| 睡眠の質（7.5hとの差） | 色の調和度・円の大きさ |
| 連続記録日数 | 特殊エフェクト（黄金の輝き、虹色リングなど） |

### ギャラリー
- カレンダー形式で過去の作品をサムネイル表示
- クリックで拡大表示 + 睡眠データの詳細確認
- 月ごとの切り替え

### 実績システム（10種類）
| 実績名 | 条件 |
|-------|------|
| はじめての夜 | 初めて睡眠を記録する |
| アーリーバード | 22時前に就寝する |
| ナイトオウル | 深夜2時以降に就寝する |
| パーフェクトスリープ | 7〜8時間の理想的な睡眠 |
| ぐっすり | 9時間以上眠る |
| ウィークウォリアー | 7日連続で記録 |
| 2週間マスター | 14日連続で記録 |
| アートコレクター | 10作品を集める |
| ギャラリーマスター | 30作品を集める |
| スリープチャンピオン | 30日連続で記録 |

## 技術スタック

- **HTML / CSS / JavaScript**（バニラJS、フレームワーク不使用）
- **Canvas API**（ジェネラティブアート描画）
- **Web Audio API**（アラーム音生成）
- **Notification API**（ブラウザ通知）
- **localStorage**（データ永続化）

## 使い方

1. `sleep-art-gallery/index.html` をブラウザで開く
2. 「おやすみ」ボタンを押す → 目覚まし時刻を設定（任意）
3. 翌朝「おはよう」ボタンを押す → アート作品が生成される
4. 「ギャラリー」タブで過去の作品を閲覧
5. 「実績」タブで達成状況を確認

## 注意事項

- データはブラウザの localStorage に保存されます（ブラウザのデータを消去すると記録も消えます）
- 目覚まし機能はブラウザのタブを開いたままにする必要があります
